# Album Cover Image Download Guide

## Quick Method: Spotify/Apple Music

1. **Open Spotify Web Player** or **Apple Music** in your browser
2. Search for each album
3. **Right-click** on the album cover
4. Select **"Copy Image Address"** or **"Inspect Element"**
5. For Spotify: Look for URLs like `https://i.scdn.co/image/...`
6. For Apple Music: Look for URLs in the page source
7. Download the image and save to `assets/` folder with the correct filename

## Bandcamp Method (Best Quality)

2. **Standing on the Corner - Red Burns**: https://standingonthecorner.bandcamp.com/album/red-burns
3. **Blu - Her Favorite Color**: https://blu.bandcamp.com/album/her-favorite-color
4. **Blue Iverson - Hotep**: Search Bandcamp
5. **Knxwledge - Anthology**: https://knxwledge.bandcamp.com/album/anthology
6. **Fly Anakin & Koncept Jack$on - Chapel Drive**: Search Bandcamp
7. **Navy Blue - Ada Irin**: Search Bandcamp

**To download from Bandcamp:**
- Right-click the album cover
- Select "Save Image As..."
- Save to `assets/` folder

## Google Images Method

1. Search: `"[Album Name] album cover"`
2. Click **Tools** → **Size** → **Large**
3. Find the highest quality image
4. Right-click → **Save Image As...**
5. Save to `assets/` folder with correct filename

## Testing Without Images

The sketch will work without images - it will show colored rectangles with album names as placeholders. This is perfect for testing the pattern, rhythm, and movement features!

## Image Format

- **Format**: JPG or PNG (both work)
- **Recommended Size**: At least 500x500 pixels (larger is better)
- **Aspect Ratio**: Square (1:1) works best
