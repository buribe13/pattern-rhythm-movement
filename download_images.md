## Method 2: Manual Download

1. Visit each artist's Bandcamp page or official website
2. Right-click on the album cover image
3. Select "Save Image As..."
4. Save to the `assets` folder with the correct filename

## Method 3: Using Google Images

1. Search for "[Album Name] album cover"
2. Use Tools > Size > Large
3. Download the highest resolution image
4. Rename and save to the `assets` folder

## Alternative: Use Placeholder Images

If you want to test the sketch immediately, you can create placeholder images or the sketch will show colored rectangles with album names until the images are loaded.
